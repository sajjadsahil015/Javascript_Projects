# Quickstart: Weather Dashboard MVP

## Prerequisites
- A modern web browser (Chrome, Firefox, Edge)
- An OpenWeatherMap API Key (Free Tier)

## Installation
1. Clone the repository.
2. Navigate to `Weather-Dashboard/`.
3. Open `script.js` and replace `YOUR_API_KEY` with your actual key.
4. Open `index.html` in your browser.

## Usage
1. **Search**: Enter a city name (e.g., "Tokyo") and click "Search".
2. **View**: Observe current weather and 5-day forecast.
3. **Persist**: Refresh the page; the last searched city will reload automatically.

## Development
- **HTML**: Structure in `index.html`.
- **Styles**: Design in `style.css`.
- **Logic**: API calls and UI updates in `script.js`.
- **Tests**: Manual verification against User Stories in `spec.md`.
